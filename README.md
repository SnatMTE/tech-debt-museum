# Tech Debt Museum

**A curated collection of old code, experimental projects, and digital odds & ends.**

Everything in this repo is the stuff that didn't have a home elsewhere — assignments from college and university, quick experiments born out of boredom, half-baked ideas, and the occasional "it worked at the time" prototype. Think of it less like a portfolio and more like an archaeological dig through my own coding history.

## What's Inside

| Language | Projects |
|----------|----------|
| **Python** | Budget app, polygon area calculator, rock-paper-scissors, traffic lights, hangman, tic-tac-toe, and more |
| **C#** | ATM simulator, tic-tac-toe, Facebook auto-commenter |
| **PHP** | Facebook auto-poster, loot stealer (don't ask) |
| **Assembly** | Traffic lights |

## Why Does This Exist?

Not every piece of code needs to be polished, production-ready, or portfolio-worthy. Some of it was written at 2 a.m. to meet a deadline, some of it was me figuring out a concept, and some of it exists just because I was curious. This repo is where all that code lives on instead of being lost to a forgotten hard drive.

Oh, and **Loot Stealer** — despite its sketchy name - was just a basic loot management system for *Final Fantasy XIV*. It eventually turned into a full-blown application that uses SQLite. What you're seeing in this repo is back when the poor thing was just a simple webhook app.

## Disclaimer

This code is presented as-is, for historical and entertainment purposes only. It likely doesn't follow best practices, may have security holes, and probably won't run without some tinkering. **You probably shouldn't use any of this in production.**
